# Product Requirements Document: ContPAQi AI Bridge Sales Website

## 1. Overview

### 1.1 Product Name
ContPAQi AI Bridge - Sales and Marketing Website

### 1.2 Product Vision
Create a modern, conversion-optimized sales website to market and sell ContPAQi AI Bridge - an AI-powered invoice processing desktop application that automates data entry for Mexican accountants using ContPAQi accounting software.

### 1.3 Target Audience
- Mexican accountants and bookkeepers
- Accounting firms handling multiple clients
- Small to medium businesses with in-house accounting
- CFOs and financial managers looking to improve efficiency

### 1.4 Value Proposition
"Eliminate 90% of manual invoice data entry. Let AI extract, validate, and post invoice data directly to ContPAQi - while you maintain full control."

## 2. Technical Foundation (Already Implemented)

The website uses the following pre-configured stack:
- Next.js 14 with App Router and TypeScript
- Tailwind CSS with custom brand colors
- PostgreSQL with Prisma ORM
- NextAuth.js v5 for authentication
- Stripe for payments and subscriptions
- next-intl for internationalization (EN/ES)
- Resend for transactional emails

## 5. API Endpoints (Complete Implementation)

### 5.1 Checkout API (POST /api/checkout)
- Accept plan and billing interval
- Create Stripe checkout session
- Return checkout URL

### 5.2 Webhook Handler (POST /api/webhooks/stripe)
- Handle checkout.session.completed
- Handle subscription events
- Handle invoice events
- Create license on successful payment
- Send confirmation email

### 5.3 License APIs
- POST /api/license/validate - Validate license key
- POST /api/license/activate - Activate on machine
- POST /api/license/deactivate - Deactivate machine
- GET /api/license/status - Get license status

### 5.4 Contact API (POST /api/contact)
- Store contact submission
- Send notification email
- Return success response

### 5.5 Auth APIs (NextAuth.js)
- Signin/signout
- Session management
- Password reset
